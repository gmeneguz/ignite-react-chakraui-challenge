## Desafio Chakra UI - World Trip Site

**Objetivos do desafio**

- Chakra UI
- Responsividade
- NextJs

## Como executar

**yarn server é necessário para iniciar Json Server com dados fictícios**

```sh
yarn
# Iniciar o servidor json
yarn server
# Iniciar o servidor o app next
yarn dev
```
