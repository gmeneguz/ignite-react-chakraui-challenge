## Chakra UI Challenge - World Trip Site

**Challenge goals**

- Chakra UI
- Responsiveness
- NextJs

## How to run

**yarn server is required in order to start Json server with mock data**

```sh
# Install dependecies
yarn
# Start json server
yarn server
# Start app in dev mode
yarn dev
```
